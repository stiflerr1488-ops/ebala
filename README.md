# Project Images

SVG images generated in the requested format for the following words:

- Активность → `images/aktivnost.svg`
- Календарь → `images/kalendar.svg`
- Правила → `images/pravila.svg`
- Состав → `images/sostav.svg`
- Отчеты → `images/otchety.svg`
- Фидбек → `images/fidbek.svg`
- Магазины → `images/magaziny.svg`

To regenerate, run:

```bash
python generate_images.py
```
